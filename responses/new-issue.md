# 微信小程序开发

# 工具介绍

kbone 是一个致力于微信小程序和 Web 端同构的解决方案。

[微信官方文档-Kbone](https://developers.weixin.qq.com/miniprogram/dev/extended/kbone/)


## Quick Start:

```bash
# 安装
npm install -g kbone-cli
```
​
```bash
# 创建项目
kbone init my-app

# 开发小程序端
npm run mp

# 开发 Web 端
npm run web

# 构建 Web 端
npm run build
```
